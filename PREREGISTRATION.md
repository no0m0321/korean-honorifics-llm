## 1. 연구 질문

프롬프트의 한국어 경어(공손도)가 LLM의 (a) 과제 성능과 (b) 출력 스타일에 영향을 주는가? 그 영향은 모델 계열·과제 유형·경어의 종류(상대/주체/객체)에 따라 어떻게 달라지는가? (Yin et al., 2024의 영·중·일 연구를 한국어로 확장)

## 2. 가설

### (A) 진성 사전 등록 — 상대 높임 (실험 전 고정, git 2026-06-10)
- **H1.** 경어 단계가 높아져도 정확도는 단조증가하지 않는다.
- **H2.** 경어 단계는 출력 스타일(길이·어휘 다양성·출력 어체)에 영향을 미친다.
- **H3.** 과제 유형(분류·추론·생성)에 따라 최적 경어 단계가 다르다.
- **H4.** 경어 단계 × 모델 상호작용이 유의하다(한국어 특화 모델의 민감도 차이).
- **탐색적.** 해라체 명령형이 거부·격식 붕괴를 유발하는가; 사어체(하게·하오) OOD 처리.

### (B) 계획된 확장 — 주체·객체 높임 (1축 실험 후 추가)
- **H5.** 주체·객체 높임은 상대 높임과 직교한다(교호 무유의).
- **H6.** 굴절(-시-) ≫ 보충법(드리다/여쭙다): 생성적 형태소를 어휘적 보충법보다 잘 처리한다.
- **H7.** 입력 -시-는 출력으로 전파(미러링)되며 강도는 모델별로 다르다.
- **H8.** 보충법 처리 능력은 한국어 어휘 지식의 함수(특화 모델·큰 모델에서↑).
- **H9.** 과잉존대보다 과소생성 쪽으로 치우친다.
- **탐색적.** 경어 적절성 판정(HAJ) 능력; 의뢰자 래퍼의 전파(NSMC 채널2).

## 3. 설계

- **독립변수 1**: 상대 높임 6단계(해라/해/하게/하오/해요/하십시오), 단계당 패러프레이즈 3종(템플릿 아티팩트 통제).
- **독립변수 2**: 모델 3종 — EXAONE 3.5 7.8B(한국어 특화), Qwen3-8B(다국어), Llama 3.1 8B(영어 중심). 전부 로컬 Ollama 4bit, greedy(temperature 0).
- **독립변수 3 (확장)**: 주체 높임(±), 객체 높임(±). 측정 직교를 위해 상대 높임은 해요체로 고정, 높임 대상은 항상 제3자.
- **과제**: NSMC 감정분류(1,000) · KoBEST COPA 추론(1,000) · 뉴스요약(100); 확장 과제 HSC(216)·memo(80)·HAJ(30)·NSMC채널2(200).
- **표본 규모 근거**: 검정력 시뮬레이션으로 사전 결정(n=1,000, 통합분석 Δ4%p에서 power 0.93).

## 4. 종속변수
정확도(Accuracy / ROUGE-L), 응답 길이(어절·형태소), 어휘 다양성(Distinct-1/2·TTR), 출력 어체(Kiwi 어말어미 규칙 분류), 입출력 어체 전이, 거부·파싱 실패율, (확장) 주체/객체 높임 생성·전파율·과잉/과소생성.

## 5. 통계 분석 계획 (실험 전 고정)

| 분석 | 방법 |
|---|---|
| 정확도 단계 간 | Cochran's Q → 사후 pairwise McNemar |
| 단계 × 모델 상호작용 (H4) | 로지스틱 혼합효과모형 `correct ~ level*model + (1\|item)` (1순위), 수렴 실패 시 GEE/glmer 대안 |
| 연속 지표 | 선형 혼합효과, 비모수 대안 Friedman |
| 등가성 | TOST (고빈도 단계쌍) |
| 다중비교 | Holm-Bonferroni |
| 모든 평균 | 부트스트랩 95% CI |

> 대안 조항: 사전 등록한 혼합효과모형(statsmodels VB)이 수치적으로 퇴화할 경우 GEE(교환가능, 군집=아이템) 및 R lme4::glmer로 교차검증한다. (실제로 발동되었으며 논문 7.5절에 보고)
